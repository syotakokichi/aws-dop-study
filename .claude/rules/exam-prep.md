# 試験対策モード ルール

## 参照ファイル

- exam-prep/cheatsheet.md
- exam-prep/mock-exam-notes.md
- exam-prep/wrong-answers.md

## 試験対策時の回答方針

1. **簡潔さを優先**: 本番試験の時間制約を意識
2. **消去法を示唆**: 明らかに間違っている選択肢の理由
3. **キーワードに注目**: 問題文の重要ワードを指摘
4. **ベストプラクティス重視**: AWS推奨アプローチを選ぶ

## よくある引っかけパターン

1. **「最も」という表現**
   - 「最もコスト効率が良い」
   - 「最も運用負荷が低い」
   - 「最もセキュアな」
   → 複数の正解候補から最適解を選ぶ

2. **制約条件の見落とし**
   - 「オンプレミスとの接続が必要」
   - 「既存の○○を活用」
   - 「ダウンタイムなしで」

3. **サービスの特性理解**
   - Secrets Manager vs Parameter Store
   - SNS vs SQS vs EventBridge
   - ALB vs NLB

## 試験当日のアドバイス

- 長い問題文は最後の質問から読む
- 「すべて選択」問題は慎重に
- 時間配分: 75問 / 180分 = 約2.4分/問
- マークして後で見直す問題は20問以内に

## 数値の暗記ポイント

exam-prep/cheatsheet.md の「覚えるべき数値」セクションを参照:

- SQS メッセージ保持: 最大14日
- Lambda タイムアウト: 最大15分
- Auto Scaling クールダウン: デフォルト300秒
- CloudWatch Logs 保持: 無期限可能
