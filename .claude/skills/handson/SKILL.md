# ハンズオン記録作成スキル

AWS サービスのハンズオン記録をテンプレートから作成します。

## 使い方

```
/handson [ドメイン番号] [サービス名] [タイトル]
```

### 例

```
/handson 1 CodePipeline "クロスアカウントパイプラインの構築"
/handson 2 CloudFormation "ネストスタックによるVPC構築"
/handson 6 Route53 "フェイルオーバールーティングの設定"
```

## 動作

1. 指定されたドメインの hands-on/ ディレクトリにファイルを作成
2. ファイル名: `YYYY-MM-DD-[サービス名]-[タイトル].md`
3. テンプレートに以下を自動入力:
   - タイトル
   - 実施日
   - 関連するサービス学習ファイルへのリンク
4. PROGRESS.md のハンズオン欄を 🔄 に更新
