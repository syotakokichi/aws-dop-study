# 用語クイズ生成スキル

CLAUDE.md の用語集やサービス学習ファイルから用語クイズを生成します。

## 使い方

```
/quiz [問題数] [タイプ]
```

### タイプ

- `term`: 英語→日本語の用語変換
- `concept`: 概念の説明を問う
- `compare`: サービス/機能の比較
- `scenario`: シナリオベース（簡易版）
- `mixed`: 上記をミックス（デフォルト）

### 例

```
/quiz 10              # 10問のミックスクイズ
/quiz 5 term          # 5問の用語クイズ
/quiz 8 compare       # 8問の比較クイズ
```

## 動作

1. CLAUDE.md の用語集を読み込む
2. 各ドメインのサービス学習ファイルからキーコンセプトを抽出
3. 指定されたタイプと問題数でクイズを生成
4. インタラクティブに回答を受け付け
5. 正答率を表示

## クイズ形式

### 用語クイズ (term)

```
Q1. 「Blue/Green Deployment」の日本語訳と説明として正しいものは？

A. ブルー/グリーンデプロイ - 2つの同一環境を切り替えるデプロイ方式
B. カナリアデプロイ - 少数のユーザーに先行リリース
C. ローリングアップデート - 段階的にインスタンスを更新
D. イミュータブルデプロイ - 新しいインスタンスを作成して切り替え
```

### 概念クイズ (concept)

```
Q2. CloudFormation の「ドリフト検出」について正しい説明は？

A. テンプレートと実際のリソース構成の差異を検出する機能
B. スタック間の依存関係を検出する機能
C. リソースのコスト変動を検出する機能
D. セキュリティ脆弱性を検出する機能
```

### 比較クイズ (compare)

```
Q3. Parameter Store と Secrets Manager の違いとして正しいものは？

A. Secrets Manager のみ自動ローテーション機能がある
B. Parameter Store の方が保存できる値のサイズが大きい
C. Parameter Store のみ暗号化をサポートする
D. Secrets Manager は無料で使用できる
```

### シナリオクイズ (scenario)

```
Q4. EC2インスタンスにSSH鍵を使用せずにセキュアに接続したい場合、
最も適切なサービスは？

A. Systems Manager Session Manager
B. EC2 Instance Connect
C. AWS Direct Connect
D. VPN Gateway
```

## 学習効果

- 反復学習による記憶定着
- 試験で問われる形式に慣れる
- 弱点の特定（間違えた問題の分野を記録）

## 正答率の活用

間違えた問題は自動的に記録され、次回のクイズで重点的に出題されます。
