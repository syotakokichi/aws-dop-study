# 間違えた問題の分析

## 分析方法

1. 間違えた問題を記録
2. 正解と自分の回答を比較
3. なぜ間違えたか分類
4. 関連知識を復習

## 間違いの分類

| カテゴリ | 説明 | 対策 |
|----------|------|------|
| 知識不足 | そもそも知らなかった | 該当サービスを学習 |
| 理解不足 | 知っていたが理解が浅かった | 深堀り学習 |
| 読み違い | 問題文を誤読した | 問題文を丁寧に読む |
| 勘違い | 類似サービスと混同 | 比較表を作成 |
| ケアレス | 単純ミス | 見直し時間を確保 |
